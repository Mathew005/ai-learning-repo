# AI Learning Roadmap

This repository follows a structured roadmap for mastering AI Systems Engineering.
Each folder represents a distinct phase of learning.

## Structure

*   `01_Fundamentals`: LLM APIs, Prompt Engineering.
*   `02_Basic_RAG`: Retrieval Augmented Generation basics.
*   `03_Advanced_RAG`: Advanced RAG techniques (LangChain, Graph).
*   `04_Agents`: AI Agents (ReAct, Tool use).
*   `05_Multi_Agent_Systems`: Orchestration (CrewAI, AutoGen).
*   `06_Optimization`: Evaluations and Deployment.

## Usage
Navigate to a folder to begin that phase.
